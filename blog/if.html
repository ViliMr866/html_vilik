if

Příkaz if slouží k vytváření podmínek. Program díky němu dokáže rozhodovat, co udělá v různých situacích. Podmínky jsou základní součástí téměř každého programu.

Například lze kontrolovat správnost hesla, věk uživatele nebo porovnávat čísla.

vek = 20

if vek >= 18:
print("Máš přístup")

Pokud je podmínka pravdivá, vykoná se odsazený kód. Pokud není pravdivá, program pokračuje dál.
