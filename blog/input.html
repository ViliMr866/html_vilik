input ( )

Funkce input() slouží ke komunikaci programu s uživatelem. Díky ní může program přijímat data napsaná člověkem pomocí klávesnice. Bez této funkce by program pouze vypisoval informace, ale neuměl by reagovat na odpovědi uživatele.

Nejčastěji se input() používá při zadávání jména, věku, hesla nebo různých číselných hodnot. Program nejprve zobrazí otázku a potom čeká, dokud uživatel něco nenapíše a nepotvrdí klávesou Enter.

jmeno = input("Jak se jmenuješ? ")

print("Ahoj", jmeno)

Je důležité vědět, že input() vždy vrací textový řetězec. Pokud tedy chceme pracovat s čísly, musíme použít převod pomocí int() nebo float().
